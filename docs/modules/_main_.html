getPixels

  • getPixels(data: string): Array<string> | undefined
    • Defined in main.ts:125

    Takes XBM data as a string and tries to extract the data portion of the image by looking for a block of hex values delimited by curly braces. If found, the block is split using the comma character as delimiter and the hex values are converted to binary strings which are padded to the length of a byte. This is then converted to an array of characters and returned. If no block matching the conditions if found, undefined is returned.

    Parameters

    • data: string

      XBM formatted data as a string

    Returns Array<string> | undefined

    An array containing the data portion of the input as binary strings
